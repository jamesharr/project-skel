# Project Skeletons
Some project skeletons James uses to speed up getting started.

## Assumptions:
* OSX/Linux with `make` installed.
* VSCode as the editor
* Appropriat

## Usage
```sh
cd ~/workspace
git clone git@github.com/jamesharr/project-skel

cp -a project-skel/SKEL new-project
cd new-project
code .
make
```
